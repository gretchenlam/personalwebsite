# Portfolio Website
Welcome to my portfolio site! This site was created using ReactJS and implements Swiper for sliding components as well as EmailJS for contact services.

### Project Features:
- Responsive Layout
- Fixed Navigation
- Pop-up Modals
- Tab Panels
- Slider + Pagination
- Contact Form
- Scroll to Top
